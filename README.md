## 🎮 인재원 edition 콘솔 게임 실행 방법 🎮

### 1. 게임 폴더 열기

다운로드한 게임 폴더를 엽니다.

폴더 안에 아래 형식의 파일이 있는지 확인하세요.

```
파일명.py
```

---

### 2. 명령 프롬프트(cmd) 실행하기

키보드에서:

```
Windows 키 + R
```

을 누릅니다.

입력창에:

```
cmd
```

입력 후 **Enter**를 누릅니다.

---

### 3. 게임 폴더로 이동하기

아래 명령어를 입력합니다.

```
cd 게임폴더경로
```

예시:

```
cd Desktop\PythonGame
```

---

### 4. 게임 실행하기

아래 명령어를 입력합니다.

```
python 파일명.py
```

Enter를 누르면 게임이 시작됩니다.

---

### 🖥️ 화면 지우기

화면이 지저분해졌다면:

```
cls
```

를 입력하세요.

화면이 깨끗하게 정리됩니다.

---

### ⌨️ 자주 사용하는 명령어

| 명령어              | 기능          |
| ---------------- | ----------- |
| `python 파일명.py` | 게임 실행       |
| `cls`            | 화면 지우기      |
| `dir`            | 현재 폴더 파일 확인 |
| `exit`           | cmd 종료      |

---

### ❗ 실행이 안 될 때

**Python이 설치되어 있는지 확인**

입력:

```
python --version
```

정상이라면:

```
Python 3.x.x
```

처럼 표시됩니다.

---

### 게임 종료

게임 안의 종료 메뉴를 이용하거나:

```
Ctrl + C
```

를 누르세요.

---
